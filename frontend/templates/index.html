Path Finder

Este sistema utiliza o algoritmo de Dijkstra, um método da área de pesquisa operacional, para encontrar o caminho mais curto entre dois pontos em um grafo ponderado. O algoritmo funciona explorando todas as rotas possíveis a partir do ponto de partida, atribuindo custos a cada aresta e selecionando o caminho com o menor custo total até o ponto de chegada.
